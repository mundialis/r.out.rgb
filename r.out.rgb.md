## DESCRIPTION

*r.out.rgb* is a GRASS GIS addon python script to export a RGB raster as
GeoTiff and to return a SLD for the RGB composite.

## EXAMPLE

```sh
r.out.rgb red=red green=green blue=blue output=rgb_output.tif 
```
